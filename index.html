    Firefox OS


    Também conhecido como B2G (Boot to Gecko), o Firefox OS é uma plataforma para dispositívos móveis.


    Seu alicerce é constituído pelo HTML5 e pelas Web APIs.


    Liberdade de plataformas proprietárias. Escreva uma vez, execute onde quiser!

    Gaia


    O Gaia é a interface gráfica com o usuário no Firefox OS.


    Tudo que você precisa saber para criar aplicativos são tecnologias web como JavaScript, HTML e CSS.


    As APIs de acesso são definidas pela W3C DAP (DeviceAPis).

    Gecko


    Motor de renderização que está presente em diversos navegadores {Firefox, SeaMonkey, Camino etc}


    Camada de execução de serviços. Interação entre o Gaia e as APIs.


    Gonk


    Sistema Operacional low-level, Kernel Linux e Camada de abstração de Hardware


    Kernel baseado no AOSP (Android Open Source Project)


    É uma camada de portabilidade. Isso é, fornece uma interface entre hardware e o Gecko.

    Media - WebRTC


    • WebRTC permitirá realizar o envio de vídeo e áudio em tempo real, de navegador para navegador.
    • Um usuário usando Chrome será capaz de realizar chamada em tempo real para um usuário no Firefox.
    • O projeto está na reta final, mas ainda possui algumas limitações.
    • Disponível no Firefox 22 em diante, Chrome 25 e Opera 18.

      • ASCII Camera

    Media - WebVTT


    • Elementos de vídeo podem possuir uma tag <track>
    • A tag <track> define uma legenda que será carregada ao exibir o vídeo (formato .vtt)
    • Simplicidade de formatação e similaridade com .srt
      •                             <video src="media/demo.webm" controls preload="auto">
                                <track kind="subtitles"
                                       src="captions/demo.vtt"
                                       label="ptsubs"
                                       srclang="pt-BR"
                                       id="demotrack" default />
                            </video>

    • É uma API em JavaScript para renderização interativa de modelos 2D e 3D.

    SVG

    • SVG é uma linguagem XML, similar ao XHTML, que pode ser utilizada para criar gráficos e imagens.
    • Formato aberto, livre de direitos e criado pela W3C.
    • Oferece inúmeros benefícios se comparados a imagens normais.
    • Nossa logo é um SVG!
